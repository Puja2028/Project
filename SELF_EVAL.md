# Self-Evaluation Checklist

Mark each item **yes** or **no** after you complete (or attempt) the task.

## Baseline questions

| Category | Question | Self-Eval (yes/no) |
|----------|----------|-------------------|
| Repo discovery | Can you inspect an unfamiliar repo and produce a class/service/module inventory in 30 minutes? | |
| Data model | Can you produce an ER diagram from code, migrations, ORM models, or schema files in 45 minutes? | |
| API mapping | Can you map all API endpoints to handlers/controllers in 30 minutes? | |
| Flow tracing | Can you trace one endpoint/event/cron flow end-to-end across files in 45 minutes? | |
| Testing | Can you identify the test framework, relevant test files, and exact test command for a module in 30 minutes? | |
| Greenfield build | Can you build a small FastAPI service from scratch with tests in 60 minutes? | |
| Node.js build | Can you build a small Node.js service/CLI with tests in 60 minutes? | |
| Rust build | Can you build a small Rust CLI/library with tests in 60 minutes? | |
| Parallel work | Can you split a task across multiple worktrees or agent sessions safely? | |
| Verification | Can you separate what the agent suggested from what you manually verified? | |
